How translation happens

Animated view of the SQL to LINQ pipeline: parse, bind, rewrite, and emit.

1. Parse SQL

Read SQL text and create a structured AST.

SELECT c.Name
FROM Customers c
WHERE c.IsActive = 1

2. Bind Meaning

Resolve table and column symbols to model entities.

  • Customers -> Customer
  • Name -> Customer.Name
  • IsActive -> Customer.IsActive

3. Rewrite Rules

Convert relational intent to LINQ operations.

  • WHERE -> .Where(...)
  • SELECT -> .Select(...)

4. Emit LINQ

Generate ready-to-use C# LINQ output.

customers
  .Where(c => c.IsActive)
  .Select(c => c.Name);
SELECT FROM WHERE

Why this view matters

Keeping this workflow isolated makes it easier for reviewers to map a bug to the exact conversion stage and open a precise issue.
