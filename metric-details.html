Trust reporting

Metric details

Detailed conversion quality, performance, and issue-coverage data from live conversion events or the latest benchmark report.

Enterprise-safe telemetry: raw SQL text and generated LINQ are not persisted. Metrics are based on status, timing, syntax target, connectivity mode, and query fingerprint metadata. Database values are telemetry tags reported by the client, not server-verified engine detection.
